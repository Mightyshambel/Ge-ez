­Ъџђ Basic Examples

Hello World

рѕЏрЅ░рѕЮ "рѕ░рѕІрѕЮ рІЊрѕѕрѕЮ!"

Variables

ріарѕхрЅ░рІІрІЇрЅЁ рѕхрѕЮ = "ріарѕі"
ріарѕхрЅ░рІІрІЇрЅЁ ріарІхрѕю = 25
ріарѕхрЅ░рІІрІЇрЅЁ ріарІЇрЅ│рѕерѕўрѕерЅЦ = ріЦрІЇріљрЅх

рѕЏрЅ░рѕЮ "рѕхрѕЮ:", рѕхрѕЮ
рѕЏрЅ░рѕЮ "ріарІхрѕю:", ріарІхрѕю
рѕЏрЅ░рѕЮ "ріарІЇрЅ│рѕерѕўрѕерЅЦ:", ріарІЇрЅ│рѕерѕўрѕерЅЦ

User Input

ріарѕхрЅ░рІІрІЇрЅЁ рѕхрѕЮ = рїЇрЅЦріарЅх("рѕхрѕЮрІјріЋ рІФрѕхрїѕрЅА: ")
рѕЏрЅ░рѕЮ "рѕ░рѕІрѕЮ", рѕхрѕЮ

­Ъћё Control Flow

If/Else Statements

ріарѕхрЅ░рІІрІЇрЅЁ ріарІхрѕю = 20

ріерѕєріљ ріарІхрѕю >= 18 {
    рѕЏрЅ░рѕЮ "ріарІІрЅѓ"
} ріарѕѕрЅарѕѕрІџрІФ {
    рѕЏрЅ░рѕЮ "рѕЇрїЁ"
}

While Loop

ріарѕхрЅ░рІІрІЇрЅЁ рЅЂрїЦрѕГ = 1

рѕ│рѕѕ рЅЂрїЦрѕГ <= 5 {
    рѕЏрЅ░рѕЮ рЅЂрїЦрѕГ
    рЅЂрїЦрѕГ = рЅЂрїЦрѕГ + 1
}

For Loop

рѕѕ рЅЂрїЦрѕГ рЅа рІѕрѕ░ріЋ(1, 6) {
    рѕЏрЅ░рѕЮ рЅЂрїЦрѕГ
}

­ЪћД Functions

Basic Function

рІўрІ┤ рѕЏрЅ░рѕЮ_рѕхрѕЮ(рѕхрѕЮ) {
    рѕЏрЅ░рѕЮ "рѕхрѕЮ:", рѕхрѕЮ
}

рѕЏрЅ░рѕЮ_рѕхрѕЮ("ріарѕі")

Function with Return

рІўрІ┤ рІерѕЏрЅБрІЏрЅх(рѕђ, рѕѕ) {
    рѕўрѕѕрѕх рѕђ * рѕѕ
}

ріарѕхрЅ░рІІрІЇрЅЁ рІЇрїцрЅх = рІерѕЏрЅБрІЏрЅх(5, 3)
рѕЏрЅ░рѕЮ "рІЇрїцрЅх:", рІЇрїцрЅх

Recursive Function

рІўрІ┤ рЇІрЅбріќрЅ║(рЅЂрїЦрѕГ) {
    ріерѕєріљ рЅЂрїЦрѕГ <= 1 {
        рѕўрѕѕрѕх рЅЂрїЦрѕГ
    } ріарѕѕрЅарѕѕрІџрІФ {
        рѕўрѕѕрѕх рЇІрЅбріќрЅ║(рЅЂрїЦрѕГ - 1) + рЇІрЅбріќрЅ║(рЅЂрїЦрѕГ - 2)
    }
}

рѕѕ рЅЂрїЦрѕГ рЅа рІѕрѕ░ріЋ(1, 11) {
    рѕЏрЅ░рѕЮ рЇІрЅбріќрЅ║(рЅЂрїЦрѕГ)
}

­ЪЈЌ№ИЈ Object-Oriented Programming

Simple Class

ріГрЇЇрѕЇ рѕ░рІЇ {
    рІўрІ┤ рѕўрїђрѕўрѕфрІФ(рѕФрѕ▒, рѕхрѕЮ, ріарІхрѕю) {
        рѕФрѕ▒.рѕхрѕЮ = рѕхрѕЮ
        рѕФрѕ▒.ріарІхрѕю = ріарІхрѕю
    }
    
    рІўрІ┤ рѕЏрЅ░рѕЮ_рѕўрѕерїЃ(рѕФрѕ▒) {
        рѕЏрЅ░рѕЮ "рѕхрѕЮ:", рѕФрѕ▒.рѕхрѕЮ
        рѕЏрЅ░рѕЮ "ріарІхрѕю:", рѕФрѕ▒.ріарІхрѕю
    }
    
    рІўрІ┤ ріарІхрѕю_рѕўрїерѕўрѕГ(рѕФрѕ▒) {
        рѕФрѕ▒.ріарІхрѕю = рѕФрѕ▒.ріарІхрѕю + 1
    }
}

# Create objects
ріарѕхрЅ░рІІрІЇрЅЁ рѕ░рІЇ_1 = ріарІ▓рѕх рѕ░рІЇ("ріарѕі", 25)
ріарѕхрЅ░рІІрІЇрЅЁ рѕ░рІЇ_2 = ріарІ▓рѕх рѕ░рІЇ("рЅЦрѕГрѕЃріЋ", 30)

# Use objects
рѕ░рІЇ_1.рѕЏрЅ░рѕЮ_рѕўрѕерїЃ()
рѕ░рІЇ_2.рѕЏрЅ░рѕЮ_рѕўрѕерїЃ()

рѕ░рІЇ_1.ріарІхрѕю_рѕўрїерѕўрѕГ()
рѕЏрЅ░рѕЮ "ріарІ▓рѕх ріарІхрѕю:", рѕ░рІЇ_1.ріарІхрѕю

Calculator Class

ріГрЇЇрѕЇ ріФрѕЇріЕрѕїрЅ░рѕГ {
    рІўрІ┤ рѕўрїђрѕўрѕфрІФ(рѕФрѕ▒) {
        рѕФрѕ▒.рІЇрїцрЅх = 0
    }
    
    рІўрІ┤ рѕўрїерѕўрѕГ(рѕФрѕ▒, рЅЂрїЦрѕГ) {
        рѕФрѕ▒.рІЇрїцрЅх = рѕФрѕ▒.рІЇрїцрЅх + рЅЂрїЦрѕГ
        рѕўрѕѕрѕх рѕФрѕ▒.рІЇрїцрЅх
    }
    
    рІўрІ┤ рѕўрЅђріљрѕх(рѕФрѕ▒, рЅЂрїЦрѕГ) {
        рѕФрѕ▒.рІЇрїцрЅх = рѕФрѕ▒.рІЇрїцрЅх - рЅЂрїЦрѕГ
        рѕўрѕѕрѕх рѕФрѕ▒.рІЇрїцрЅх
    }
    
    рІўрІ┤ рѕЏрЅБрІЏрЅх(рѕФрѕ▒, рЅЂрїЦрѕГ) {
        рѕФрѕ▒.рІЇрїцрЅх = рѕФрѕ▒.рІЇрїцрЅх * рЅЂрїЦрѕГ
        рѕўрѕѕрѕх рѕФрѕ▒.рІЇрїцрЅх
    }
    
    рІўрІ┤ рѕЏрЅ░рѕЮ_рІЇрїцрЅх(рѕФрѕ▒) {
        рѕЏрЅ░рѕЮ "рІЇрїцрЅх:", рѕФрѕ▒.рІЇрїцрЅх
    }
}

ріарѕхрЅ░рІІрІЇрЅЁ ріФрѕЇріЕ = ріарІ▓рѕх ріФрѕЇріЕрѕїрЅ░рѕГ()
ріФрѕЇріЕ.рѕўрїерѕўрѕГ(10)
ріФрѕЇріЕ.рѕЏрЅБрІЏрЅх(2)
ріФрѕЇріЕ.рѕўрЅђріљрѕх(5)
ріФрѕЇріЕ.рѕЏрЅ░рѕЮ_рІЇрїцрЅх()

File Writing and Reading

# Read from file

Lists

ріарѕхрЅ░рІІрІЇрЅЁ рЅЂрїЦрѕ«рЅй = [1, 2, 3, 4, 5]
ріарѕхрЅ░рІІрІЇрЅЁ рѕхрѕърЅй = ["ріарѕі", "рЅЦрѕГрѕЃріЋ", "рѕ░рѕІрѕЮ"]

# Access elements
рѕЏрЅ░рѕЮ рЅЂрїЦрѕ«рЅй[0]  # First element

# Modify elements

# Iterate through list

Dictionaries

ріарѕхрЅ░рІІрІЇрЅЁ рѕ░рІЇ = {"рѕхрѕЮ": "ріарѕі", "ріарІхрѕю": 25, "ріарІЇрЅ│рѕерѕўрѕерЅЦ": ріЦрІЇріљрЅх}

Simple Calculator

    ріерѕєріљ рѕхрѕФ == "+" {

    } ріарѕѕрЅарѕѕрІџрІФ ріерѕєріљ рѕхрѕФ == "*" {

            рѕўрѕѕрѕх рѕђ / рѕѕ
        } ріарѕѕрЅарѕѕрІџрІФ {
            ріарѕхрЅ░рѕІрѕЇрЇЇ "рЅарІюрѕ« рѕўріерЇІрЇѕрѕЇ ріарІГрЅ╗рѕЇрѕЮ"

    }
}

Number Guessing Game

рІўрІ┤ рІерЅЂрїЦрѕГ_рїерІІрЅ│() {
    ріарѕхрЅ░рІІрІЇрЅЁ рІерЅ░рѕ░рІѕрѕе_рЅЂрїЦрѕГ = 42
    ріарѕхрЅ░рІІрІЇрЅЁ рѕЎріерѕФрІјрЅй = 0
    ріарѕхрЅ░рІІрІЇрЅЁ ріарѕИріЊрЇі = рѕљрѕ░рЅх
    
    рѕЏрЅ░рѕЮ "рЅа 1 ріЦріЊ 100 рѕўріФріерѕЇ рІФрѕѕ рЅЂрїЦрѕГ ріарѕхрЅЦ!"
    
    рѕ│рѕѕ ріарѕИріЊрЇі == рѕљрѕ░рЅх {

        рѕЎріерѕФрІјрЅй = рѕЎріерѕФрІјрЅй + 1
        
        ріерѕєріљ рїЇрѕЮрЅх == рІерЅ░рѕ░рІѕрѕе_рЅЂрїЦрѕГ {
            рѕЏрЅ░рѕЮ "ріарѕИріЊрЇі! рЅа", рѕЎріерѕФрІјрЅй, "рѕЎріерѕФрІјрЅй"
            ріарѕИріЊрЇі = ріЦрІЇріљрЅх
        } ріарѕѕрЅарѕѕрІџрІФ ріерѕєріљ рїЇрѕЮрЅх < рІерЅ░рѕ░рІѕрѕе_рЅЂрїЦрѕГ {
            рѕЏрЅ░рѕЮ "рЅарїБрѕЮ рЅхріЋрѕй!"
        } ріарѕѕрЅарѕѕрІџрІФ {
            рѕЏрЅ░рѕЮ "рЅарїБрѕЮ рЅхрѕЇрЅЁ!"
        }
    }
}

File Processing

рІўрІ┤ рЇІрІГрѕЇ_рѕўрѕхрѕўрѕ«рЅй_рѕўрЅЂрїарѕГ(рЇІрІГрѕЇ_рѕхрѕЮ) {
